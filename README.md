# Ejercicios de Testing con Jest (JavaScript)

Ejercicios semana 5 para practicar **pruebas unitarias** con **Jest** y ver **cobertura (coverage)**.

---

## 1) Requisitos

Necesitas tener instalado:

- **Node.js**
- **npm** (normalmente se instala junto con Node)

Verifica en la terminal:

```bash
node -v
npm -v
```

## 2) Creación package.json

Ejecutar el siguiente comando para la creación del package.json

```bash
npm init -y
```

## 3) Instalación dependencias (node_modules)

Ejecutar el siguiente comando para la instalación de dependencias

```bash
npm i
```

## 4) Agregar librería jest

Ejecutar el siguiente comando

```bash
npm i -D jest
```

## 5) Ejecución coverage

Para evaluar la cobertura de los test, ejecutar el siguiente comando

```bash
npm test -- --coverage
```
